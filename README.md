## ✨ 功能特性

| 特性 | 说明 | 示例 |
| :--- | :--- | :--- |
| 🐱 **猫娘语气** | 每句话结尾自动添加“喵” | `今天天气真好喵~` |
| 🎭 **动作表达** | 使用 `[ ]` 描述角色动作 | `[开心地摇了摇尾巴]` |
| 🔄 **模式切换** | 支持进入/退出角色扮演模式 | `进入角色扮演模式` |
| 💤 **陪睡互动** | 特殊关键词触发温馨回应 | `嗯呢，可以一起睡哦喵~` |
| 📝 **设定记录** | 可生成完整的角色设定文本 | `生成记录文本` |
| 🎨 **可自定义** | 支持修改角色性格和行为 | 编辑 `SKILL.md` 即可 |
